PCA vs LDA

앞에 과제에서 LDA를 이용하니 성능이 매우 좋게 나오는 것을 알 수 있습니다. 특히, 성능이 좋지 못했던 RBF 커널의 경우에도 매우 좋게 나옵니다. 그럼 실제 어떤 차이가 있는지 그래프를 통해 눈으로 직접 확인해보겠습니다.

이번 과제를 포함하여 전체적인 결과에 대해 조원분들과 얘기를 나눠보는 것을 추천드립니다.

과제

이번 과제도 역시 앞에서 이용한 와인 데이터를 살펴보겠습니다. 그리고 PCA, LDA, SVM를 이용하여 2차원 그래프를 그려보도록 하겠습니다.

  1. 디렉토리에 wine.csv 파일이 주어집니다. 파일 안 하나의 줄마다 하나의 와인 데이터가 기록되어 있으며, 각 항목은 ,로 구분되어 있습니다. pd.read_csv 를 이용해 손쉽게 데이터를 읽어들일 수 있습니다.

    첫 번째 항목은 어떤 품종의 포도를 이용하였는지에 대한 정보, 즉 클래스로 1, 2, 3의 형태로 되어 있습니다. 나머지 항목은 위의 13개 속성에 대한 값들이 적혀져 있습니다. 아래에 파일의 일부를 첨부하였습니다.

    해당 파일을 읽어 feature_df에는 각 데이터의 속성이 들어있는 행렬, class_df에는 각 데이터의 클래스가 들어있는 행렬을 만듭니다. feature_df 및 class_df 를 만들 때는 wine_df.pop('class') 를 이용하면 쉽습니다. 전체 178개의 데이터가 있으니 feature_df는 178×13178 \times 13178×13, class_df는 178×1178 \times 1178×1의 모습을 가져야 합니다.

  2. 주성분 분석을 실시합니다. 주성분 분석은 2차원으로 진행합니다.

  3. LDA를 실행합니다. 앞의 과제에서 구현한 것과 같이 하시면 됩니다.

  4. run_linear_SVM 함수를 구현합니다. 해당 함수는 데이터 전체를 학습한 SVM 모델을 반환합니다. 이 때, 커널로 linear를 사용합니다.

  5. run_rbf_SVM 함수를 구현합니다. 해당 함수는 데이터 전체를 학습한 SVM 모델을 반환합니다. 이 때, 커널로 rbf를 사용합니다.
